LOESS

LOESS can be described mathematically as a sequence of local weighted least-squares fits.

1. Data setup

Suppose you observe data

\[ (x_1,y_1), (x_2,y_2), \dots, (x_n,y_n). \]

For each target point \(x_0\), LOESS fits a small regression using points near \(x_0\), giving nearby points more weight.

2. Local regression model

At a target location \(x_0\), LOESS approximates the regression function locally by a low-degree polynomial.

Local linear LOESS

The most common form is local linear:

\[ y_i \approx \beta_0 + \beta_1(x_i-x_0). \]

Local quadratic LOESS

A slightly more flexible version is local quadratic:

\[ y_i \approx \beta_0 + \beta_1(x_i-x_0) + \beta_2(x_i-x_0)^2. \]

Notice the polynomial is written in terms of \((x_i-x_0)\), so the fit is centered at the target point \(x_0\).

3. Weighted least squares

The coefficients are chosen by minimizing a weighted sum of squared errors.

Local linear case

\[ (\hat\beta_0,\hat\beta_1) \arg\min_{\beta_0,\beta_1} \sum_{i=1}^n w_i(x_0)\Big[y_i-\beta_0-\beta_1(x_i-x_0)\Big]^2. \]

Local quadratic case

\[ (\hat\beta_0,\hat\beta_1,\hat\beta_2) \arg\min_{\beta_0,\beta_1,\beta_2} \sum_{i=1}^n w_i(x_0)\Big[y_i-\beta_0-\beta_1(x_i-x_0)-\beta_2(x_i-x_0)^2\Big]^2. \]

The fitted LOESS value at \(x_0\) is then

\[ \hat f(x_0)=\hat\beta_0, \]

because when \(x_i=x_0\), all the \((x_i-x_0)\) terms vanish.

4. The weights

The weights depend on how far \(x_i\) is from \(x_0\).

A standard LOESS weight is the tricube weight:

\[ w_i(x_0)= \begin{cases} \left(1-\left|\dfrac{x_i-x_0}{h(x_0)}\right|^3\right)^3, & |x_i-x_0|<h(x_0),\\ 0, & |x_i-x_0|\ge h(x_0). \end{cases} \]

Here:

  • \(h(x_0)\) is the distance from \(x_0\) to the furthest point in the local neighborhood
  • points outside that neighborhood get weight 0
  • nearby points get larger weights
  • points very close to \(x_0\) get weights near 1

5. How the span enters

If the span is \(s\), then at each \(x_0\), LOESS uses roughly the nearest

\[ m = \lceil sn \rceil \]

points.

So:

  • smaller \(s\) means a smaller neighborhood, more wiggle
  • larger \(s\) means a larger neighborhood, smoother fit

For example, with span = 0.1, each local fit uses about 10% of the data.

6. Matrix form

For a local linear fit at \(x_0\), define

\[ X(x_0)= \begin{pmatrix} 1 & x_1-x_0\\ 1 & x_2-x_0\\ \vdots & \vdots\\ 1 & x_n-x_0 \end{pmatrix}, \qquad y= \begin{pmatrix} y_1\\y_2\\ \vdots\\ y_n \end{pmatrix}, \qquad W(x_0)=\operatorname{diag}(w_1(x_0),\dots,w_n(x_0)). \]

Then the weighted least-squares estimator is

\[ \hat\beta(x_0)= \big(X(x_0)^T W(x_0) X(x_0)\big)^{-1} X(x_0)^T W(x_0)y. \]

The LOESS fit at \(x_0\) is the first component:

\[ \hat f(x_0)=\begin{pmatrix}1 & 0\end{pmatrix}\hat\beta(x_0). \]

For the quadratic version, just add a third column \((x_i-x_0)^2\).

7. Robust LOESS

Some versions of LOESS add a second layer of weights to reduce the effect of outliers.

After an initial fit, compute residuals

\[ r_i = y_i - \hat f(x_i). \]

Then define robust weights, often using Tukey’s bisquare form:

\[ \delta_i= \begin{cases} \left(1-\left(\dfrac{r_i}{6M}\right)^2\right)^2, & |r_i|<6M,\\ 0, & |r_i|\ge 6M, \end{cases} \]

where \(M\) is typically the median absolute residual.

Then the new weighted least-squares problem uses combined weights

\[ w_i(x_0)\ \delta_i. \]

So outliers get downweighted.

8. Final conceptual formula

LOESS does not give one single global formula like

\[ y = a + bx + cx^2. \]

Instead, for each \(x_0\), it defines

\[ \hat f(x_0) = \text{the fitted value from a weighted local polynomial around }x_0. \]

So the fitted curve is built point by point.

9. In words

At each target \(x_0\):

  1. choose nearby points,
  2. weight them by distance from \(x_0\),
  3. fit a local linear or quadratic regression,
  4. take the fitted value at \(x_0\),
  5. repeat for many \(x_0\)’s.

That sequence of local fits is the LOESS curve.
